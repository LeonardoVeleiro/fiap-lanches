FIAP Lanches
Descrição do Problema

    Atualmente, as cantinas universitárias enfrentam grande dificuldade com a alta demanda em curtos intervalos. Isso gera filas extensas, aumento do tempo de espera e estresse para os alunos, que possuem pouco tempo para se alimentar. Além disso, a falta de mão de obra encarece a operação e dificulta o atendimento eficiente.

Solução Proposta

    O FIAP Lanches é um aplicativo mobile que permite que alunos e colaboradores realizem pedidos e pagamentos antecipadamente. Com isso, elimina-se a necessidade de filas, melhora-se a organização da cozinha e otimiza-se o tempo do usuário, que apenas retira o pedido quando estiver pronto.

Google Colab (execução do projeto)

Como Executar
Copie o código para o Google Colab ou um ambiente Python local
Execute todas as células ou o script
Ao final, execute:
srs.relatorio()
O sistema exibirá o relatório completo dos requisitos

Estrutura do Projeto
Prioridade: Enum para classificação de prioridade
RequisitoFuncional: Estrutura dos requisitos funcionais
RequisitoNaoFuncional: Estrutura dos requisitos não funcionais
SRS: Classe principal que gerencia os requisitos e gera relatórios

Funcionalidades Implementadas
Cadastro e autenticação de usuários via RM
Visualização de cardápio categorizado
Detecção de localização por GPS
Gestão de estoque diário
Pagamentos via cartão, Pix e vale-alimentação
Notificações de status do pedido
CRUD de cardápio para gerentes
Controle de acesso (RBAC)
Relatórios de vendas
Configuração de horário de funcionamento
Organização de pedidos por ordem de chegada (FIFO)
